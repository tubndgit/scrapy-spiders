Required:
- Python 2.7
- Scrapy installed
- Selenium installed
==========================================
- Extract this package
- Config params in config.json file
- Open command window(Windows) or Terminal(Linux) then move to this folder
- Run command: 
    scrapy crawl collect -a proxy_opt=5
- The data will be downloaded and stored under folder that set in config.json file
- That's all
